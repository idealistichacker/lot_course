# 队友交接包（当前版本）

## 1. 固件版本信息
- 当前版本：v0.2（无树莓派阶段）
- 固件文件位置：esp32_firmware/main.ino
- 编译环境：VS Code + Arduino Community Edition
- 依赖库：DHT sensor library by Adafruit、Adafruit Unified Sensor

## 2. 硬件映射
- DHT11 DATA: GPIO2
- 摄像头映射：见核心板和扩展板参数文档

## 3. 数据协议
- 文档位置：docs/数据协议与联调规范.md
- 当前协议版本：v0.2（新增 camera_ok 字段）

## 4. 测试结果
- 采集状态：稳定（DHT11 连续多轮 OK）
- 上传状态：稳定（连续多轮 POST 200，ack_seq 连续递增）
- 摄像头状态：Camera init OK（当前为初始化检测，不上传图片）
- 已定位并修复问题：
	- 热点频段错误导致 Wi-Fi 超时（已切换 2.4GHz）
	- SSID 字符导致联调复杂（已改为英文 SSID=esp32）

## 5. 队友接入步骤（树莓派）
1. 按协议实现 /upload 接口
2. 返回统一 ACK（ok, ack_seq, server_time）
3. 验证 seq 对齐和重试逻辑
4. 在树莓派侧接入图像字段（下一阶段 C 方案：multipart + JPEG）
5. 在树莓派侧接入 VLM 与 RAG 决策流水线

## 6. 待办与风险
- 待完成项：
	- 摄像头 JPEG 实际上传（当前仅 camera_ok）
	- 树莓派端 VLM 识别与 RAG 推理
	- Webhook 告警闭环
- 风险项：
	- 手机热点稳定性与 IP 漂移
