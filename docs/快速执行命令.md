# 快速执行命令

## 1. 启动本地假后端
在目录 mock_receiver 下执行：

```powershell
pip install -r requirements.txt
uvicorn server:app --host 0.0.0.0 --port 8000
```

## 2. 查电脑热点内网 IP（Windows）
```powershell
ipconfig
```
找到无线网卡在手机热点下的 IPv4 地址，例如 192.168.43.200。

## 3. 配置 ESP32
- 复制 esp32_firmware/secrets.example.h 为 esp32_firmware/secrets.h
- 修改 WIFI_SSID, WIFI_PASSWORD, UPLOAD_URL

## 4. 烧录并观察串口
- 波特率：115200
- 关键成功日志：
  - DHT11 OK
  - WiFi connected
  - Upload success
  - Entering deep sleep

## 5. 本地接口手工验收
```powershell
curl -Method POST -Uri http://127.0.0.1:8000/upload -ContentType "application/json" -Body '{"device_id":"esp32s3-cam-01","fw_version":"0.1.0","seq":1,"temp_c":25.1,"rh_pct":80.2,"dew_point_c":21.1,"wifi_rssi":-60,"sensor_ok":true}'
```
